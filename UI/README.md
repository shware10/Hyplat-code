# UI

게임 UI 연출, 인터랙션, 애니메이션 및 상태 표시를 담당하는 스크립트 모음입니다.

## Scripts

* **UIManager** : 게임 상태에 따라 UI 전환 및 텍스트 연출(페이드, 스케일, 바운스)을 관리하는 클래스 

* **ButtonManager** : 버튼 텍스트/이미지의 hover, 클릭, 흔들림 등의 인터랙션을 관리하는 매니저 

* **ClickableText** : 텍스트 UI에 마우스 이벤트를 전달하는 클래스 

* **ClickableImage** : 이미지 UI에 포인터 이벤트를 전달하는 클래스 

* **BlinkingText** : 텍스트를 일정 간격으로 깜빡이게 하는 연출 클래스 

* **IntroManager** : 인트로 연출 및 씬 전환을 관리하는 클래스 

* **BoxHandler** : 자식 오브젝트에 흔들림 애니메이션을 적용하는 클래스 

* **ChildsWobbler** : 자식 오브젝트들을 sin 기반으로 흔들리게 하는 클래스 

* **CanvasScaler** : 카메라 뷰에 맞게 Canvas 크기와 위치를 조정하는 클래스 

* **LifeHandler** : 플레이어의 하트(UI 체력)를 관리하는 클래스
