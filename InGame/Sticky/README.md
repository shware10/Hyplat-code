# Sticky

스티키 오브젝트의 생성, 동작, 연출 및 이벤트 반응을 담당하는 스크립트 모음입니다.

## Scripts

* **Sticky** : 스티키의 공통 동작과 이벤트 구독(Fit/Unfit, GameState)을 관리하는 베이스 클래스 

* **SpawnedSticky** : 생성된 스티키의 등장/사라짐 연출 및 상호작용을 처리하는 클래스 

* **StickySpawner** : 오브젝트 풀링을 통해 스티키를 생성하고 랜덤 위치에 배치하는 클래스 

* **BB** : 특정 스티키의 애니메이션 및 상태 기반 연출을 담당하는 클래스 

* **Tail** : LineRenderer를 이용해 스티키의 꼬리 트레일을 생성하는 클래스
