# Managers

게임 전반의 상태, 오디오, 이펙트, 생성 로직 등을 관리하는 매니저 클래스 모음입니다.

## Scripts

* **GameManager** : 게임 상태, 점수, 생명, 전체 게임 흐름을 관리하는 핵심 매니저 
* **AudioManager** : BGM 및 SFX를 관리하고 게임 상태에 따라 오디오를 제어하는 클래스 
* **FeverEffecter** : 피버 상태에서 파티클 이펙트를 순차적으로 재생하는 클래스 
* **RandomPosGenerator** : 화면 영역을 분할하여 랜덤 위치를 생성하는 유틸성 매니저
